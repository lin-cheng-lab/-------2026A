## 主要内容

### 问题一（基于Paper 4方法）
- 建立20方程20未知量的刚体力学模型
- 考虑12m/s、24m/s、36m/s三种风速
- 分析锚链沉底和悬浮两种状态

### 问题二（基于NSGA-III算法）
- 三目标优化：最小化吃水深度、钢桶倾角、游动半径
- 获得70个Pareto最优解
- 最优方案：重物球3000kg，锚链18m，游动半径减少32.5%

### 问题三（基于Paper 4方法）
- 考虑水流力影响（1.5m/s）
- 多参数综合优化：锚链型号、长度、重物球质量
- 水深16-20m范围内的鲁棒性分析
